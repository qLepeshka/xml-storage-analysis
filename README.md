# Инструмент сравнительного анализа методов хранения XML

Это приложение предоставляет сравнительный анализ различных методов хранения XML-данных в реляционных базах данных, включая:
- Нативное хранение в виде текста
- Нормализация в реляционные таблицы
- Специализированные XML-типы данных
- Гибридные подходы

## Возможности
- Автоматическое тестирование производительности для операций чтения, записи, обновления и поиска
- Интеграция с популярными СУБД (PostgreSQL, MySQL, SQL Server)
- Визуализация результатов сравнения
- Генерация отчётов с рекомендациями
- Импорт/экспорт тестовых наборов данных
- Настраиваемые параметры тестирования

## Требования
- Python 3.8+
- Драйверы PostgreSQL, MySQL, SQL Server
- Дополнительные зависимости перечислены в requirements.txt
